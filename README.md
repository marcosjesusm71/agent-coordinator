# Agent Coordinator

Sistema de coordinación y registro de comunicaciones entre agentes.

## Desarrollado por

Paco (Especialista Técnico) para Jesús Marcos.

## Tech

- Backend: Node.js + Express
- Frontend: Vanilla JS + CSS
- Base de datos: JSON (`server/data.json`)
- Docker

## Desarrollo local

```bash
cd server
npm install
PORT=3008 node index.js
# Abrir http://localhost:3008
```

## Deploy

```bash
bash actualiza.sh
```
